SafeHer AI

SafeHer AI is an AI-powered women safety platform that provides real-time protection through live location sharing, SOS alerts, fake calls, AI voice assistance, and guardian monitoring. The platform enables users to instantly connect with trusted guardians during emergencies while leveraging AI for proactive safety.

Installation

Clone Repository

git clone https://github.com/yourusername/safeher-ai.git
cd safeher-ai

⸻

Install Frontend

cd frontend
npm install

⸻

Install Backend

cd ../backend
npm install

⸻

Environment Variables

Backend (.env)

PORT=5000
MONGODB_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET
GOOGLE_CLIENT_ID=YOUR_CLIENT_ID
GOOGLE_CLIENT_SECRET=YOUR_CLIENT_SECRET
FRONTEND_URL=http://localhost:5173

⸻

Frontend (.env)

VITE_API_URL=http://localhost:5000/api
VITE_GOOGLE_CLIENT_ID=YOUR_CLIENT_ID

⸻

Running Project

Backend

cd backend
npm run dev

⸻

Frontend

cd frontend
npm run dev

Application runs at:

Frontend:
http://localhost:5173

http://localhost:5000

Android App (Capacitor)

Install Capacitor

npm install @capacitor/core @capacitor/cli
npm install @capacitor/android

Initialize

npx cap init

Build

npm run build

Add Android

npx cap add android

Copy Build

npx cap copy

Open Android Studio

npx cap open android
